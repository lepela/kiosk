키오스크 최소 도메인 (MVP)
1. Product (상품/메뉴)
- id (PK)
- name (상품명)
- price (가격)
- category (카테고리: 메인/사이드/음료)
- image_url (이미지)
- is_available (판매가능 여부)
- description (설명)
2. Order (주문)
- id (PK)
- order_number (주문번호 - 화면 표시용)
- total_amount (총액)
- status (대기/조리중/완료/취소)
- ordered_at (주문시간)
- kiosk_id (FK - 어느 단말에서 주문)
3. OrderItem (주문상품)
- id (PK)
- order_id (FK → Order)
- product_id (FK → Product)
- quantity (수량)
- price (주문 당시 가격 - 스냅샷)
4. Payment (결제)
- id (PK)
- order_id (FK → Order)
- amount (결제금액)
- payment_method (카드/현금/간편결제)
- payment_status (대기/완료/실패)
- paid_at (결제시간)
5. Kiosk (키오스크 단말)
- id (PK)
- location (위치: 매장1-1호기)
- status (활성/비활성/점검중)
- last_heartbeat (마지막 통신시간)
